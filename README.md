# Tecnologias Utilizadas
* React - Bibliteca JavaScript para criação de interfaces
* TypeScript - Tipagem segura para JavaScript
* React Hook Form - Gerenciamento do estadodo de formulários
* Zod - Validação de esquemas com TypeScript
* date-fns - Biblioteca de utilitários de data
* Lucide React - Biblioteca de Ícones
* Tailwind CSS - Framework CSS de classes utilitárias
* Tailwind Variants - Criação de variantes de estilos para componentes
* Husky - Execução de Git Hooks
* Eslint, Prettier - Qualidade do Código
* Context API - Gerenciamento de Estado da APlicação
* Service Worker - Executar o contador fora da Thread Principal
